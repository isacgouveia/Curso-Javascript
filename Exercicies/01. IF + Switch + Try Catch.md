1. Verificar se um número é par ou ímpar (IF)

Peça ao usuário um número.

- Se for par, exiba "Par".
- Se for ímpar, exiba "Ímpar".

*Resultado*
```js
    /* 1. Verificar se um número é par ou ímpar (IF) */
    const number = 12

    if(number%2 === 0 ){
    console.log("Par")
    }else{
    console.log("Impar")
    }

```

2. Classificação de idade (IF)

Receba a idade de uma pessoa e classifique:

- Menor de 12 → "Criança"
- 12 a 17 → "Adolescente"
- 18 a 59 → "Adulto"
- 60+ → "Idoso"

*Resultado*
```js
    /* 2. Classificação de idade (IF) */
    const age = 12

    if(age >= 60){
    console.log("Idoso")
    }else if(age >= 18 && age <= 59){
    console.log("Adulto")
    }else if(age >= 12 && age <= 17){
    console.log("Adolescente")
    }else{
    console.log("Criança")
    }

```

3. Menu de operações matemáticas (Switch Case)

Peça 2 números e uma operação:

- 1 = somar
- 2 = subtrair
- 3 = multiplicar
- 4 = dividir

Use switch para executar a operação.

*Resultado*
```js
    /* 3. Menu de operações matemáticas (Switch Case) */
    let firstNumber = 2
    let secondNumber = 3
    const option = 1

    switch(option){
    case 1:
    console.log("A soma é:", firstNumber + secondNumber)
    break

    case 2:
    console.log("A subtração é:", firstNumber - secondNumber)
    break

    case 3:
    console.log("A multiplicação é:",firstNumber * secondNumber)
    break

    case 4:
    console.log("A divisão é:",firstNumber / secondNumber)
    break

    default:
    console.log("Opção não encontrada!")
    }

```

6. Cálculo com erro tratado (Try/Catch)

Solicite dois números e mostre a divisão.

- Use try/catch para evitar divisão por zero.
- Se erro, exibir "Erro: divisão inválida".

*Resultado*
```js
    /* 6. Cálculo com erro tratado (Try/Catch) */
    firstNumber = 1
    secondNumber = 0

    try{
    if(isNaN(firstNumber) || isNaN(secondNumber)){
        throw"Um dos valores não é um número!"

    }
    
    if(secondNumber === 0){
        throw "Não é possível dividir por zero!"
    }

    let result = firstNumber / secondNumber
    console.log("Resultado da divisão é:", result)

    }catch(error){
    console.log(error)
    }

```

9. Cálculo de imposto com IF + Try/Catch

Receber o valor de um produto.

- Se valor < 100 → imposto 5%
- 100 a 500 → 10%
- 500 → 15%
- Use try/catch para garantir que o valor informado seja numérico.

*Resultado*
```js
    try{
    let input = 90
    let result

    if(isNaN(input)){
        throw "O valor informado não é númerico!"
    }

    if(input){
        result = input * 0.15
    }else if(input >= 100 && input <= 500){
        result = input * 0.10
    }else{
        result = input * 0.05
    }

    console.log("Valor do produto: R$ " + input)
    console.log("Imposto cálculado: R$ " + result.toFixed(2))
    console.log("Valor total com imposto: R$ " + (input+result).toFixed(2))

    }catch(error){
    console.log(error)
    }

```
