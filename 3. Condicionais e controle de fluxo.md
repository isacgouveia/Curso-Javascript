## Estrutura de Condição
Permite executar diferentes ações **com base em uma condição** (Verdadeira ou falsa).

As estruturas de condição são utilizadas para a tomada de decisões, permitindo que o fluxo de execução seja direcionado com base em condições.

*exemplo:*
```js
    let isOpen = true

    if(isOpen === true){
        console.log("Está aberta!")
    }else{
        console.log("Está fechada!")
    }
```

## Operador Condicional Ternário
Operador condicional ternário é uma forma concisa de escrever uma instrução.
De **IF Else** por **? Verdadeiro : Falso**

*Exemplo:*
```js
    let isAdmin = true
    console.log(isAdmin === true ? "Permitido!" : "Negado!")

    if(isAdmin){
        console.log("Permitido!")
    }else{
        console.log("Negado!")
    }

    let age = 16
    console.log(age >= 18 ? "Permitido!" : "Negado!")

    if(age >= 18){
    console.log("Permitido!")
    }else{
        console.log("Negado!")
    }
```

## Condicionais

- **IF**

Estrutura condicional que executa a afirmação dentro do bloco, se determinada condição for verdadeira.

*Exemplo:*
```js
    // Estrutura do IF
    if(CONDIÇÃO){
        Instrução
    }

    let isOpen = true
    if(isOpen){
        console.log("Conexão aberta!")
    }

    let hour = 13
    if(hour >= 13){
        console.log("Bom dia!")
    }

    if(hour <= 12){
        console.log("Bom dia!")
    }

```
**Observação:** a terceira situação onde hour <= 12 não irá retornar nenhuma informação, pois na estrutura do IF só exibe o que for verdadeiro. 

<a id="page4"></a>
-  **IF ELSE**

O **ELSE** é uma estrutura complementar ao IF, que é acionada quando a condição do IF não é atentida.

*Exemplo:*
```js

    let age = 17
    if(age <= 18){
        console.log("Você não pode dirigir!")
    }else{
        console.log("Você pode dirigir!")
    }

    age = 28
    if(age <= 18){
        console.log("Você não pode dirigir!")
    }else{
        console.log("Você pode dirigir!")
    }
```
Caso o age for maior que 18 ativa o ELSE, demonstrado no segundo exemplo.

<a id="page5"></a>
- **ELSE IF**

No javascript, o **ELSE IF** é uma estrutura de condição é usada para testar múltiplicas condições sequencialmente. 

*Exemplo:*
```js
    let hour = 11

    // DE
    if(hour <= 12){
        console.log("Bom dia!")
    }

    if(hour > 19){
        console.log("Boa noite")
    }

    if(hour > 12){
        console.log("Boa tarde!")
    }

    // Para
    
    if(hour <= 12){
        console.log("Bom dia!")
    }else if(hour > 19){
        console.log("Boa noite")
    }else if(hour > 12){
        console.log("Boa tarde!")
    }

    // Segundo exemplo
    if(hour <= 12){
        console.log("Bom dia!")
    }else if(hour > 19 && hour <= 18>){
        console.log("Boa tarde")
    }else{
        console.log("Boa noite!")
    }
```

- **Switch Case**

A condição switch aalia uma expressão, combinando o valor da expressão para um cláusula case, e executa as instruções associadas ao case.

*Exemplo:*
```js
    // Estrutura
    switch(expressão){
        case valor1:
            Instruções
            break
    }
    
    let option = 1

    switch(option){
        case 1:
            console.log("Consultar pedido")
            break
        
        case 2:
            console.log("Falar com atendente")
            break

        default:
            console.log("Opção Inválida!")
    }

```
**Observação:** podemos utilizar uma estrutura do switch que se chama DEFAULT, caso nenhuma condição for atentida ele vai exibir o default.

## Tratamento de Exceções
Estrutura de condição para lidar com erros inesperados que interrompem o fluxo normal da aplicação. Exemplos incluem tentar acessar um arquivo inexistente ou conectar-se a um banco de dados indisponível. 

<img align="center" height="400" src="https://i.imgur.com/ExxvVAU.png"  />

*Exemplo:*
```js
    console.log(result) = UNCAUGHT REFERENCE ERROR

    // Estrutura do Try Catch

    try {
        // TENTA executar algo

    } catch {
        // CAPTURA o erro para tratar.
    }

    try {
        console.log(result)

    } catch (error){

        console.log("Não foi possível executar o seu pedido!")
        console.log(error)
    } finally {
        console.log("Resposta independentemente do resultado")
    }

    // Segundo exemplo

    let result = 0
    try{
        if(result === 0){
            throw new Error("O valor é igual a zero")
        }
    }catch (error){
        console.log(error)
    }


```
**Observação:** 
- no catch (){} pode inserir qualquer nome!
- O bloco Finally é uma forma de executar a instrução independentemente do resultado. (opcional)
- throw new Error() - excessões personalizadas.
