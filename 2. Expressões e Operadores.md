## Expressões e Operadores

### <a id="page2">Operadores</a>
**Operadores são símbolos** que realizam operações em operandos (valores ou variáveis).
São utilizados para manipular e comparar valores, realizar operações aritméticas, lógicas, de atribuição, entre outras.

### <a id="page3">Expressão</a
Uma expressão **é uma combinação** de valores, variáveis, operadores e chamadas de função que, quando avaliada, resulta em um valor.

```js
    // Expressão aritmética que soma dois números
    let sum = 5 + 3 + 7

    // Expressão aritmética que cálcula a média
    let average = sum / 3 
```

*exemplo:*
```js
    // Operadores artiméticos
    
    console.log("Soma:", 12 + 8) = Soma: 28

```

## Incremento e Decremento

*exemplo:*
```js
    let number = 10
    console.log(number) = 10

    number = number + 1
    console.log(number) = 11

    number++ // Incremento
    console.log(number) = 12

    // Incrementa após, irá mostrar o número 13 no console seguinte
    console.log(number++) = 12 
    console.log(number) = 13

    // Incremento antes, mostra o número 14 no próprio console
    console.log(++number) = 14

    number-- // Decremento
    console.log(number) = 13

    // Reduzindo o código
    number += 1 /* number = number + 1 */
    console.log(number) = 14

    number -= 2 /* number = number - 2 */
    number *= 2 /* number = number * 2 */
    number /= 2 /* number = number / 2 */
    number %= 2 /* number = number % 2 */
```

## Ordem de Precedência
Quando uma expressão contém múltiplos operadores, na programação e em expressões matemática, a ordem de precedência **define qual operação será realizada primeiro**.

|Nível|Categoria|Operadores|
|---;|;---;|---;|
|7(alto)|exponenciação|**|
|6|multiplicação|*,/,//,%*|
|5|adição|+,-|
|4|relacional|==,!=,<=,>=,>,<|
|3|lógico|not|
|2|lógico|and|
|1|lógico|or|

```js
    let value = 2+3*2/2
    console.log(value)

    /*
    Resposta:
    3*2 = 6
    6 / 2 = 3
    3 + 2 = 5
    */

   let value2 = (2+3)*3/2
   console.log(value2)

    /*
    Resposta:
    2+3 = 5 
    5 * 3 = 15
    15/2 = 7.5
    */

```

## Grouping Operator
O uso do Group Operator, representado pelos parenteses, é utilizado para definir a ordem de procedência das operações.

*Exemplo:*
```js
    let total1 = 1 + 2 + 3
    console.log(total1) = 6

    let total2 = 1 + (2-3) + 2
    console.log(total2) = 5
    /*
    (2 = 3) = 1
    1 + 1 + 3 5
    */

```

## Operador Lógico

- == (igual a)
*Exemplo:*
```js
let one = 1
let two = 2

console.log(one == two) = FALSE
console.log(one == 1) = TRUE
console.log(one == "1") = TRUE

```
A terceira situação deu TRUE pois, o igual a (==) verifica apenas o conteúdo e não o tipo de string.

- != (diferente de)

*Exemplo:*
```js
    let one = 1
    let two = 2

    console.log(one != two) = TRUE
    console.log(one != 1) = FALSE
    console.log(one != "1") = FALSE
```
A terceira situação deu TRUE pois, o diferente de (!=) verifica apenas o conteúdo e não o tipo de string.

- === (Estritamente igual)

*Exemplo:*
```js
    let one = 1
    let two = 2

    console.log(one === two) = FALSE
    console.log(one === 1) = TRUE
    console.log(one === "1") = FALSE
```
A terceira situação deu FALSE pois, o estritamente igual (===) verifica tanto o conteúdo quanto o tipo de string.

- !== (Estritamente Diferente)

*Exemplo:*
```js
    let one = 1
    let two = 2

    console.log(one !== two) = TRUE
    console.log(one !== 1) = FALSE
    console.log(one !== "1") = true
```
A terceira situação deu FALSE pois, o estritamente diferente (!==) verifica tanto o conteúdo quanto o tipo de string.

## Operadores de lógicos
Operadores lógicos são essenciais para tomada de decisões em aplicações.
Operadores lógicos AND, OR e NOT.

- **AND (&&)** - retorna **VERDADEIRO** se toda a análise for **VERDADEIRA**.
- **OR (||)** - retorna **VERDADEIRO** se pelo menos uma for **VERDADEIRO**.
- **NOT (!)** - utiliza para deixar a variável com o valor oposto da declarada.

```js
    let email = true
    let password = true

    // AND (E) &&
    console.log(email && password) = TRUE

    password = false
    console.log(email && password) = FALSE

    // OR (OU) ||
    console.log(email || password) = TRUE

   // NOT (negação) ! 
    console.log(!password) = TRUE
```
